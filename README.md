# SmileOffice

**Desc:** 文档编辑器

**Author:** ZSS

---

### 项目说明

> 这是一个文档编辑器项目，基于开源软件`ONLYOFFICE Docs Community Edition`进行二次开发，实现对文档的创建、编辑、分享、查看，多人协同等操作...



###  开源说明

系统 100%开源，本软件遵循 MIT 开源协议



### 功能介绍

1. 用户可以新建文档、表格、幻灯片等文件
2. 用户可以上传自己的文件
3. 可对文件进行在线看查看和编辑
4. 支持文件重命名
5. 统计所拥有的各类文档数量
6. 收藏文档 -- not yet
7. 可将文件分享他人 -- not yet
8. 可以共同编辑 -- not yet
9. ......
